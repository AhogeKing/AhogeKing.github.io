第一章: 初识C语言

C语言是一种通用的、结构化的编程语言，诞生于20世纪70年代初，由丹尼斯·里奇在贝尔实验室开发。C语言设计简洁、功能强大，并且能高效地与硬件进行交互，因此被广泛应用于操作系统、嵌入式系统、编译器和各类应用软件的开发。

尝试C最简单的方法是在浏览器中访问 https://www.onlinegdb.com。在那里，我们遵行下列步骤就可以完成第一个入门的程序。

1.删除编辑区中的任何代码，然后输入：

例子

#include <stdio.h>
int main() {
printf("Hello, world!" );
return 0;
}


2.单击绿色的run按钮, 这样应该就能看到Hello, world!显示在网页底部。恭喜你，刚刚成功运行了第一个C程序!

3.也可以尝试单击{}Beauty按钮，它将自动格式化你的代码。(其实就是美观点)

让我们来简单地解释上面的代码。


#include <stdio.h>: 这一行代码是一个预处理指令，用来包含标准输入输出库的头文件stdio.h。这个头文件中包含了我们要使用的输入输出函数，比如printf函数。通过包含这个头文件，我们就可以在程序中使用这些函数了。

int main() {: 这行代码定义了程序的主函数main。在C语言中，程序的执行是从main函数开始的。int表示这个函数会返回一个整数类型的值，通常是0或1。main函数的花括号 {}表示函数体的开始和结束。

printf("Hello, world!");: 这一行代码使用了printf函数来向屏幕打印输出"Hello, world!"这段文字。printf是一个标准库函数，用于格式化输出。这里的"Hello, world!"是一个字符串常量，会被直接输出到终端窗口。

return 0;: 这一行代码表示main函数的返回值是0。在C语言中，return 0;通常表示程序成功结束。如果你在命令行或其他环境中运行这个程序，返回0代表程序已经正常运行完毕，没有错误。

常见的C文件布局

#include <stdio.h>
int main() {

return 0;
}

我们新手在每次写程序前，可以先将此布局默下来，当然，默不下来也不要紧，随着练习次数的增加，自然而然就记下来啦。

实践出真知

我们可以通过可以破坏我们的程序来了解C程序必须遵循的原则。“Hello，world!”程序为例，试着进行以下的更改之一，然后编译运行它。看看会发生什么!
如果你这样做... ...它会失败, 因为...
#include <stdio.h> 在没有包含stdio.h头文件的情况下，编译器无法识别 printf函数
重命名main函数，如: func 编译器找不到程序的入口
缺少分号 ; ;是C语句结束的标志，缺少它编译器将无法正确识别语句的结束
缺少花括号{} 编译器无法确定代码块的开始和结束

准备代码编辑软件

虽然在线的代码编辑网站很方便，但是它的功能和个性化还是有一定的限制，学好编程语言还是得有一款本地好用的软件。 于是，我推荐大家下载Sublime Text。下载完后，我们就可以在Sublime Text中编写程序了，并在本地编译运行。

简单的C程序

#include <stdio.h>
int main() {
int age;
age = 18;

printf("我是一名");
printf("大学生\n");
printf("我的岁数是%d\n", age);

return 0;

}

光看代码也许并不知道打印的具体内容，所以，编译并运行该程序以查看结果吧。首先，用Sublime Text新建一个文件，并 以.c作为扩展名。例如: first.c
大家可以猜一下运行结果是什么。

如果一切正常运行，该程序的输出应该是：
我是一名大学生
我的岁数是18

让我们来逐行分析上面的代码。
int age;: 声明了一个int(整型)的变量，名为age。
age = 18;: 把数值18赋给变量age
两个printf之间是不会有空格或者换行的，要实现换行的功能，就要在 字符串中添加转义字符\n
%d: 当printf遇到 %d时，它会寻找后续参数中对应的整数值，并将其转换为字符串格式进行 输出。
